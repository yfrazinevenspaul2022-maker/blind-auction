# Blind Auction 🔨

## Description
A command-line blind auction program built with Python as part of the
**100 Days of Code** challenge (Day 10). Multiple bidders can enter their
bids privately and the highest bidder wins.

## Features
- Multiple bidders can participate
- Bids are hidden from other bidders
- Screen clears between bidders for privacy
- Automatically finds and announces the winner

## How to Use
1. Run main.py
2. Enter your name
3. Enter your bid amount
4. Answer yes if there are more bidders
5. Repeat until all bidders have placed their bid
6. Winner is announced automatically

## Example

    Enter your name: Alice
    Enter your bid: 150
    Are there more bidders? yes or no: yes
    Enter your name: Bob
    Enter your bid: 200
    Are there more bidders? yes or no: no
    The highest bidder is: Bob, with the highest bid: 200.0
